我们的主循环在算法 1 中总结。我们首先模拟高度场流体。重力、外部力和边界条件在 2.1 节 中讨论。然后，我们使用标准模拟包来模拟刚体和软体。接着，将固体和高度场流体耦合，如 2.3 节 所述。下一步，计算固体上的浮力、阻力和升力等流体力并施加。然后修改流体高度场和速度场以考虑固体的运动。之后，生成粒子来替代高度场无法解析的液体。我们再模拟这些粒子的运动并将它们落回高度场（见 2.4 节）。最后，我们用额外的位移贴图渲染高度场，以表示比模拟网格分辨率更短波长的小波。溅射粒子用屏幕空间技术渲染（2.5 节）。

2.4. 粒子生成与模拟

我们在模拟器中采用三类粒子：喷溅 (spray)、溅射 (splash) 和 泡沫 (foam)。

  • 喷溅粒子 和 溅射粒子 表示从高度场中分离出来的液体部分。
  • 喷溅粒子通常是小而快速移动的液滴；
  • 溅射粒子则表示剩余的大体积水体；
  • 泡沫粒子代表漂浮在水面上的泡沫。

在我们的模拟框架中，喷溅、溅射和泡沫粒子可以由 破碎波、瀑布、以及固体与流体的相互作用 自发生成。除此之外，它们还可以通过用户定义的外部源显式产生，例如 水龙头出水、倾倒液体或降雨。

在本工作中，我们没有建模水下气泡上升的现象。如果需要，可以结合 [TSS*07] 提出的方式进一步扩展。

2.5.2. FFT 波动平流 (FFT Waves Advection)

由于实时性约束，高度场模拟无法解析波长小于网格分辨率 Δx\Delta xΔx 的小波。
直接减小 Δx\Delta xΔx 会增加网格数和时间步需求，代价过大。

因此我们在高度场上叠加 FFT 小波模拟 [MWM87, Tes99]，其特性为：

  1. 随流速平流；
  2. 不会过度扭曲；
  3. 受拉伸过大会消失；
  4. 计算开销小。

方法：

  • 使用 FFT 生成高分辨率的波浪纹理 F(s,t)F(s,t)F(s,t)，只包含高频波；
  • 在高度场网格点上计算纹理坐标和权重，并双线性插值至像素，用于 bump mapping 或网格位移；
  • 在船只示例中，渲染网格分辨率提升至 14Δx\tfrac{1}{4}\Delta x41​Δx，对比图 7 展示了无 FFT、恒定权重 FFT 与变化权重 FFT 的差别。

3. 实验结果与讨论 (Results and Discussion)

我们在 CPU 与 GPU (CUDA) 上实现了本文方法。

  • 测试平台：Intel Core i7 2.67GHz，4GB 内存，NVIDIA GTX480。
  • 刚体、软体和布料模拟使用 NVIDIA PhysX SDK。
  • 网格大小与粒子数量列于表 1。
  • 时间步长：除船只示例外均为 Δt=16.66\Delta t=16.66Δt=16.66ms，船只示例为 Δt=20\Delta t=20Δt=20ms。

  • CPU 情况下，高度场与粒子模拟均为多线程实现。

方法局限性

  1. 体积守恒问题：
    高度积分步骤保持体积守恒，但耦合与过冲修正部分则不然。我们发现体积损失在实际效果中并非关键；如有需要，可采用 [Hes07] 的全局质量修正方法。

  2. 稳定性问题：
    我们的稳定性增强手段不能保证无条件稳定，但在所有示例中，能够以合理时间步进行显式积分。

未来工作

  • 瑞利射流 (Rayleigh Jet)：当致密物体完全浸入水中时，水面塌陷会产生二次喷溅（Rayleigh jet）。目前我们正在实验一种方法来模拟这一现象。
  • 渲染融合：未来工作还包括如何将由高度场与粒子共同表示的水体实现无缝且高效的渲染。
